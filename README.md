# Sharan Marulsidh Biradar — Developer Portfolio

This is the source code for my personal portfolio website, built as **Deliverables 1 & 2** of my web development journey. It features a modern, terminal-inspired aesthetic with a strong focus on web fundamentals, advanced responsive architecture, and accessibility.

## 🚀 Live Demo
**[View Live Site](https://sharanbiradar20-hash.github.io/portfolio/)** *(Update this link once your GitHub Pages site is live)*

## ✨ Key Features (Deliverables 1 & 2)

- **Semantic HTML5 (Deliverable 1):** Built using strict, proper semantic tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`) for a crystal clear document structure.
- **WCAG Accessibility (Deliverable 1):** 
  - Fully keyboard-navigable (including mobile menu and form inputs).
  - ARIA attributes, roles, and labels for screen readers.
  - Skip-to-content links.
  - Accessible form validation with `aria-live` announcements and proper focus handling.
- **Advanced CSS3 & Responsive Architecture (Deliverable 2):**
  - **CSS Grid & Flexbox:** Complex two-dimensional layouts using Grid, and localized component alignment using Flexbox.
  - **Mobile-First Responsive Design:** Flawless adaptation across mobile, tablet, and desktop viewports using media queries and fluid clamp functions.
  - **CSS Custom Properties (Variables):** A complete, maintainable design system powering a dynamic **Light & Dark Mode** theme toggle.
  - **Premium Aesthetics:** Terminal-style UI with rich micro-animations, glassmorphism scroll effects, and hover transitions.
- **SEO Optimized:** Complete meta tags, Open Graph tags, Twitter cards, and JSON-LD structured data for every page.
- **Vanilla JavaScript:** Zero dependencies. Custom logic for theme toggling, mobile navigation, and client-side form validation.

```bash

cd portfolio
```
